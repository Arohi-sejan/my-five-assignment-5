Question-2: What is the difference between innerHTML, innerText, and textContent ?

1. innerHTML: Returns or sets the HTML content (including HTML tags) of an element. Can be used to get or change the entire HTML structure inside an element.
2. innerText: Returns the visible text content of an element (ignores HTML tags). It is influenced by the styling of the element, e.g., if text is hidden by CSS, it won't be included. 3. textContent:
Returns the text content of an element, including all text inside child elements. Does not consider any styling or hidden text; it gets all the text, regardless of visibility or formatting.

Question-3 What is event delegation in the DOM?

Event delegation is a technique in the DOM where instead of adding event listeners to individual elements, you add a single event listener to a parent element. This listener handles events for all child elements, even if they are added dynamically after the event listener is set.

Key points:

  • Reduces memory usage by not attaching event listeners to every child element.
  • Works by leveraging event bubbling, where an event starts at the target element and bubbles up to the parent.
  • Efficient for handling events on dynamically created elements.

Question-4 What is event bubbling in the DOM?

Event Bubbling is an event propagation mechanism where an event starts at the target element and then propagates upward to its parent, grandparent, and so on until it reaches the root element.

Key Points:

  • The event first occurs on the target element and then bubbles up through its ancestors.
  • By default, most events in the DOM bubble unless explicitly stopped using event.stopPropagation().
  • Event bubbling is useful for event delegation, making it efficient to handle multiple child elements with a single event listener.

Question-5: How do you create, add, and remove elements using JavaScript?

Creating, Adding, and Removing Elements in JavaScript

  1. Creating an Element
  2. Use document.createElement() to create a new element.

Adding an Element

  • Use appendChild() to add the element as a child of a parent element.
  • Use append(), prepend(), before(), and after() for more flexible insertion.

Removing an Element

  • Use removeChild() on the parent element to remove a specific child.
  • Use remove() directly on the element to remove itself.
